# IoT-Device-Companion-App
Full-stack web app to run, manage and monitor experiments on a prototype IoT device.

### Topics
React, TypeScript, Node.js, Express.js, MongoDB, IoT

### Prerequisites
1. make sure node.js is installed,
2. npm is installed,
3. clone repository

## Server
#### Configure
```
cd node_server
npm install
```
#### Run
```
node server.js
```

## Web app
#### Configure
```
cd react-app
npm install
```
#### Run
```
npm run dev
```
